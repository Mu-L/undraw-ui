## Toast属性

| 属性     | 说明     | 类型                                    | 默认值 |
|----------|---------|-----------------------------------------|--------|
| message  | 消息文字 | string                                  | -      |
| type     | 消息类型 |'normal' / 'success' / 'warning' / 'info' /'error' | normal
| duration | 显示时间 | number                                  | 2000   |
